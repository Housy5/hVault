<h1>hVault</h1>
<p>
  hVault is an offline application for windows to store and encrypt files.
</p>
<h2>Features</h2>
<ul>
  <li>GUI-based vault management.</li>
  <li>Self-contained local storage.</li>
  <li>Easily import, export and open files.</li>
  <li>Very organizable with custom folders.</li>
  <li>AES encryption.</li>
  <li>Search functionality.</li>
  <li>Drag and drop supported</li>
</ul>
